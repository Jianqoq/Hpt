sqrt

sqrt(x: &Tensor<T>) -> Result<Tensor<C>, TensorError>

Compute x\sqrt{x}x​ for all elements

Parameters:

x: Input values

Returns:

Tensor with type C

Examples:

use hpt::{FloatUnaryOps, Tensor, TensorError};

fn main() -> Result<(), TensorError> {
    let a = Tensor::<f32>::new([4.0]);
    let b = a.sqrt()?;
    println!("{}", b);
    Ok(())
}
